# Need Engine · 需求引擎

The Nature of Needs, Desire, Motivation, Consumption & Civilizational Dynamics.

A civilization-scale, bilingual (English / 中文) atlas of **need** — reframed not as weakness but as the universal grammar of incompleteness: the gap between a conscious system and the state it requires, made into the force that drives biology, brains, markets, technology, and civilization itself. Ten interactive systems run from the chemical gradient a bacterium must climb, through dopamine and reward-prediction error, status and Maslow, the demand machine, the engineering of attention, addiction and the hedonic treadmill, meaning and transcendence, scarcity as the engine of history, AI and synthetic desire, to a unified model and a recursive engine that simulates need across eight scales — biology → emotion → social structure → economy → technology → AI attention → post-scarcity → future consciousness.

## Features

- **10 systems** with rich bilingual long-form essays
- **8-term Civilizational Demand meta-model** (survival, emotional regulation, recognition, curiosity, scarcity pressure, meaning, resource competition, identity)
- **Recursive Need Engine** — one gap-closing logic, iterated across eight scales
- **6-lens Need Analyst** (psychologist, neuroscientist, economist, systems theorist, behavioral analyst, civilization researcher)
- Live interactive visualizations: drive-emergence ladder, dopamine reward-prediction lab, morphing need hierarchy, demand-allocation economy, attention feedback-loop engine, hedonic-treadmill simulator, meaning field, scarcity-pressure sim, post-scarcity simulator
- Global EN / 中文 toggle — everything switches without reload
